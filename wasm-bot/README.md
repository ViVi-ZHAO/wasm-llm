# Chat Bot

![](./assets/preview.png)

This is a GUI based chat bot app. It allows you to run LLMs locally.

## How to use

- Git clone the repo

  ```console
  git clone https://github.com/second-state/wasm-llm
  cd wasm-bot
  ```

- Install dependencies

  ```console
  pip install -r requirements.txt
  ```

- Run the app

  ```console
  streamlit run app.py
  ```
